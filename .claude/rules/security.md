# 安全性規範

## 提交前檢查清單
- [ ] 沒有硬編碼的密鑰（API key、密碼、token）
- [ ] 所有使用者輸入已驗證
- [ ] 防止 SQL 注入（使用參數化查詢）
- [ ] 防止 XSS（輸出編碼）
- [ ] 防止 CSRF
- [ ] 沒有路徑遍歷風險
- [ ] 依賴項安全

## 機密管理
- 使用環境變數或 secret manager
- 啟動時驗證必要的機密是否存在
- 如果憑證可能洩漏，立即輪換

## 安全事件回應
1. 立即停止開發工作
2. 優先修復 CRITICAL 漏洞
3. 輪換已暴露的機密
4. 全面審查類似問題
